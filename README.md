# Titanic Data Analysis

This project analyzes the Titanic dataset to understand which factors influenced passenger survival.  
It demonstrates **exploratory data analysis (EDA), feature engineering, and visualization skills** — essential for a Data Analyst portfolio.

---

## Objective

- Analyze the Titanic dataset to identify trends affecting survival.
- Explore features such as gender, passenger class, age, family size, fare, and port.
- Use insights to tell a story about passenger survival patterns.

---

## Dataset

- Source: Titanic dataset (public dataset / Kaggle)
- Contains passenger details: Age, Sex, Class, SibSp, Parch, Fare, Port, etc.

---

## Key Steps

1. **Data Understanding** – Overview of dataset, missing values, and data types
2. **Data Cleaning** – Handle missing values and remove irrelevant columns
3. **Exploratory Data Analysis (EDA)** – Analyze survival rates by gender, class, age, fare, family size, and port
4. **Feature Engineering** – Created new features like FamilySize and IsAlone
5. **Advanced Analysis** – Combined features (e.g., Class + Gender) for deeper insights
6. **Visualization** – Bar plots, histograms, KDE plots, boxplots, and correlation heatmap
7. **Insights & Conclusion** – Summarized key trends and findings

---

## Key Insights

- Females had a significantly higher survival rate than males
- 1st class passengers survived more than 2nd and 3rd class passengers
- Children and young adults had slightly better survival chances
- Small families survived better than very large families or individuals traveling alone
- Higher fare passengers had better survival chances
- Port of embarkation had a minor effect on survival
